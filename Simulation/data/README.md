## Required metadata

For each public archive, record:

- persistent identifier and version;
- creators and source dataset citation;
- object identifier and category;
- source and processed-file checksums;
- mesh units, scale, coordinate convention, normalization, and decomposition method;
- hand model and contact/penetration-point configuration;
- generation seed and all energy/optimizer parameters;
- validation simulator build and physics settings;
- file schema and a safe loading example;
- licence and any third-party redistribution limits.

Do not commit private reviewer links, credentials, personally identifying information, or assets whose licences do not permit redistribution.
