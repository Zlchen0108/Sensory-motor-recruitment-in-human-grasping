# Code Availability

The custom simulation code used to generate and validate the robotic grasps in this study is available in this repository. The reviewer release includes the grasp-generation workflow, energy and optimization implementation, hand configuration, contact and penetration points, asset-processing utilities, curated experiment records, and a small example-object bundle. Any additional source meshes or newly generated grasp records must be provided through the access routes described in the Data Availability statement. A permanent repository URL and an archived-release DOI will be added before publication.
