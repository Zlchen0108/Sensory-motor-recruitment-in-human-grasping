# Data inventory

No study data are committed to this repository.

## Required input tables

The workflow requires four CSV or Excel tables:

1. Multi-area association-rule averages.
2. Multi-area correlation-rule averages.
3. Single-area association-rule averages.
4. Single-area correlation-rule averages.

Every table must contain a `consequents` column. The selected profile determines which additional columns are required. Rows are aligned to the following ordered consequences:

```text
TBFE, TBBE, IFFE, IFME, IFBE, TM, MFFE, MFBE, MOH, HMBE, RFFE, RFBE, LFBE, HMFE, LFFE
```
