# Code Availability

The custom code used to prepare tactile-sensor data and validate sensor-distribution patterns is available in this repository. It includes reproducible conventional machine-learning, tree-model optimization, BiLSTM, and one-dimensional CNN workflows. A permanent repository URL and a DOI for the reviewed release will be added before publication.
